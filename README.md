# 🌐 Yug Patel Portfolio

This is my personal portfolio website built using **React (Vite)**, **Tailwind CSS**, and **Framer Motion** to showcase my skills, projects, and experience as a Full Stack Web Developer.

---

## 🚀 Features

- 💻 **Responsive Design** – Works smoothly on all devices (mobile, tablet, desktop)
- 🎨 **Modern UI/UX** – Clean and minimal design
- ⚡ **Smooth Animations** – Powered by Framer Motion
- 📂 **Project Showcase** – Highlights my real-world MERN projects
- 📱 **Interactive Sections** – Skills, About, Projects, Contact
- 🔍 **SEO Optimized** – Better visibility on search engines

---

## 🛠️ Tech Stack

- **Frontend:** React.js (Vite)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons

---

## 📦 Installation & Setup

1. Clone the repository:
   ```sh
   git clone https://github.com/Yug-Patel-1228/Portfolio.git
2. Navigate to the project directory:
cd Portfolio

3. Install dependencies:
npm install

5. Start development server:
npm run dev

👉 App runs at:
http://localhost:5173/
